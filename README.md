# E-commerce Product Card UI

This project is a simple e-commerce product card built using HTML and CSS. It displays a product with an image, discount badge, and an "Add to Cart" button.

## Features
- Product image display
- Discount badge (20% OFF)
- Overlay effect on hover
- Add to Cart button
- Clean and modern card layout

## Tools Used
- HTML
- CSS

## What I Learned
- Positioning elements using CSS
- Creating overlays and hover effects
- Building reusable UI components
